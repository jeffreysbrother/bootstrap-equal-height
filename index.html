Foundation solution

another viable solution

  • no need to tediously set media query breakpoints
  • in smaller viewports, the forced equal height of the columns is retained by default, BUT we can easily counteract this action by specifying an additional attribute on the containing row (data-equalize-on-stack="false")
  • must only add two attributes to the markup

must add this markup:

  • the "data-equalizer" attribute to the containing row
  • the "data-equalizer-watch" attribute to the columns
