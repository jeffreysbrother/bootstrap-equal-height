  • no need to tediously set media query breakpoints
  • in smaller viewports, the forced equal height of the columns is NOT retained (this is likely desirable) but to see this, we must refresh the browser. This is likely not a problem for users.
  • must only add two classes to the markup (this is much easier than the CSS fix I've documented in this project)

Must add this markup:

  • an extra class to all columns being affected
  • an extra class to the row containing the desired columns
